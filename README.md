---

## ✨ Core Features

- **Dynamic Cart System**: When you click on a product, it instantly increases the cart count in the Navbar
- **Auto Clear Cart**: The cart automatically clears when you click on the "Proceed to Checkout" button
- **Instant Feedback**: Clicking the **Buy Now** button shows an “Added to cart” message using toast notification

---

## ⚙️ Technologies Used

- **React.js** (v19.2.4)
- **Tailwind CSS** + **DaisyUI**
- **JavaScript (ES6+)**
- **React-Toastify** (NPM Package for notifications)
- **JSON** (for product data)
- **Vite** (Build Tool)

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/your-username/digitools-platform.git

# Go to the project directory
cd digitools-platform

# Install dependencies
npm install

# Start the development server
npm run dev
```
